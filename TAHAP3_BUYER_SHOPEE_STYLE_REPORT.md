# Tahap 3 - Buyer Shopee Style

Perubahan dilakukan dari file Tahap 2.

## Yang dikerjakan
- Dashboard buyer dibuat lebih mobile app style seperti Shopee.
- Menu buyer di HP dibuat sebagai bottom navigation: Home, Pesanan, Profil, Belanja.
- Header mobile buyer ditambahkan dengan ikon notifikasi kanan atas.
- Badge pesanan buyer tetap muncul untuk pesanan aktif.
- Halaman beranda buyer dibuat lebih ringkas dengan hero card, shortcut cepat, statistik singkat, dan produk pilihan.
- Halaman pesanan dan profil buyer dibungkus card yang lebih rapi untuk HP.
- Produk pilihan buyer di HP dibuat 2 kolom agar tidak terlalu panjang.
- Tampilan buyer disesuaikan agar konsisten dengan tahap 1 seller dan tahap 2 admin.

## Testing
- npm install berhasil.
- npm run build berhasil.
- Tidak ada syntax error.
- Warning bundle besar masih ada, tetapi bukan error deploy.
