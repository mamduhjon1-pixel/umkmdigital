Saat admin approve seller:
- users.status berubah menjadi active
- users.role tetap seller
- seller_applications status berubah active/approved jika table ada
- seller_verifications status berubah active/approved jika table ada
- seller_wallet otomatis dibuat
- seller mendapat notifikasi
- dashboard admin realtime update
- seller setelah refresh bisa masuk dashboard seller
- seller bisa upload produk

Kalau RLS sedang disable, tetap harus jalan.
Kalau RLS nanti enable, sediakan RPC approve_seller yang aman.
